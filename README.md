# twitter-search-handson
Twitter APIのJavaラッパであるTwitter4jを使って, Twitterの検索機能を利用し画像収集をしてみましょう. 

# 実行環境
- macOS High Sierra 10.13.4
- java version "1.8.0_101"
- IntelliJ IDEA Ultimate 2018.1.3
- Gradle 4.4(Maybe)

# 依存
- Twitter4j 4.0.6
- commons io 2.6
